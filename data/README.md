# Data Layout

- `raw/`: immutable source extracts per selected dataset
- `processed/`: train/validation/test artifacts after pipeline transforms

Expected subpaths:

- `data/raw/adult_census_income/`
- `data/raw/bank_marketing/`
- `data/raw/loan_approval_classification/`
- `data/raw/ecommerce_customer_behavior/`
- `data/raw/california_housing_prices/`

Do not commit credential files or private keys.
